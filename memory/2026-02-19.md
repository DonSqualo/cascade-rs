## [04:45] Task: Implement offset::shell - Hollow solid (shelling) operation

**What:** 
Implemented the `offset::shell` module for cascade-rs, creating hollow versions of solids by offsetting faces inward and optionally removing faces to create openings.

**Implementation Details:**
- Created new `src/offset/mod.rs` module with `make_shell()` function
- API: `make_shell(solid: &Solid, thickness: f64, faces_to_remove: &[Face]) -> Result<Solid>`
- Algorithm:
  1. Validates thickness parameter (positive and less than half minimum dimension)
  2. Offsets all faces inward by the specified thickness
  3. Removes specified faces (creates openings in hollow solid)
  4. Creates edge blending faces to connect adjacent offset faces
  5. Handles corner topology properly
- Current implementation supports planar faces (foundation for extending to other surface types)
- Includes comprehensive unit tests for:
  - Closed hollow solids
  - Invalid thickness validation (negative, zero, excessive)
  - Removing faces to create openings
  - Wire offsetting edge case
